# Heritage Garden Guide

## Description
Heritage Garden Guide is a mobile-first React web application showcasing Malaysia's botanical heritage. It features a plant database, AI assistant, and guided tours, built with Tailwind CSS and Lucide React icons.

## Features
- Explore detailed plant information (traditional uses, care instructions, nutritional value).
- AI assistant for plant-related queries.
- Fully responsive design.

## Technologies Used
- React
- Tailwind CSS
- Lucide React

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hmhhmm/Heritage-Garden-Guide.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Heritage-Garden-Guide
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Start the development server:
```bash
npm start
```

Build the project for production:
```bash
npm run build
```
